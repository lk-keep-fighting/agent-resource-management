# Agent员工管理UX重构

## 背景
当前Agent管理页面对办公小白用户不够友好：
- 概念抽象（Skill/Knowledge/Agent/Prompt）
- 创建流程不清晰
- 不知如何为员工绑定技能和知识
- 不知如何快速创建不存在的技能和知识

## 需求
1. **引导式创建流程** - 分步骤引导，降低认知负担
2. **可视化技能/知识卡片** - 一目了然知道员工有什么能力
3. **一键创建关联** - 对不存在的技能/知识可直接创建并关联
4. **概念接地气** - 用"员工"、"能力"、"知识"替代抽象术语

## 设计方案

### 核心改动
1. **左侧员工列表** - 保持不变
2. **右侧改为"员工配置台"** - 引导式三步骤：
   - Step 1: 基本信息（姓名、头像、描述）
   - Step 2: 能力配置（绑定/创建技能）
   - Step 3: 知识配置（绑定/创建知识）
3. **技能/知识选择器增强** - 增加"新建"按钮，支持在选择器内直接创建
4. **顶部进度指示器** - 显示当前配置进度
